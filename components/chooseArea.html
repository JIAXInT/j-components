地区选择 ​

地区选择组件用于选择中国的省市区，支持多种配置选项。

按需引入 ​

javascript
import { createApp } from "vue";
import chooseArea from "justic-ui/chooseArea";
import "justic-ui/chooseArea/index.css";

const app = createApp();
app.use(chooseArea);

areaData地区数据object中国省市区数据
